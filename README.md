## ⚔️ 健康守護者系統 ⚔️

這不只是一個程式，這是一套**強制健康管理系統**！  
每 30 分鐘，它會：

1. 💥 **發動警告** - 彈出視窗告訴你：夠了，該動了！
2. 🌑 **黑暗降臨** - 整個螢幕變黑，進入強制休息模式
3. ⏱️ **倒數 5 分鐘** - 用大字倒數計時，讓你無處可逃
4. 🚫 **無法關閉** - 別想按 ESC，別想 Alt+F4，乖乖去走動！

**這是為了你好！** 💪

```python
🎯 技能一：時間監控

   → 置頂顯示，無法忽視

   → 想工作？門都沒有！

🎯 技能四：倒數計時
   → 大大的數字告訴你還有多久
   → 05:00 → 04:59 → ... → 自由！

🎯 終極技：循環發動
   → 休息完繼續監控
   → 永不停歇，守護健康到底！
```

## ⚙️ 自訂設定（調整戰鬥參數）

想改變時間設定？開啟 `health_reminder.py`，找到這兩行：

```python
work_interval = 30 * 60  # 工作時間（秒）- 改這裡！
lock_duration = 5 * 60   # 休息時間（秒）- 改這裡！
```

**範例魔改：**
```python
work_interval = 25 * 60  # 番茄鐘模式：25 分鐘
lock_duration = 10 * 60  # 我要休息 10 分鐘！
```
